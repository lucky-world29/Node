## How to Use  
1. Clone the repository:  
   ```bash
   git clone https://github.com/lucky-world29/Node.git
   ```
2. Navigate to the project directory:  
   ```bash
   cd Node
   ```
3. Install dependencies:  
   ```bash
   npm install
   ```
4. Run the Node.js project:  
   ```bash
   node app.js
   ```
